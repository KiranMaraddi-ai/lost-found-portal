# Lost and Found Portal (MERN Stack)

## 📌 Project Description
This is a Lost and Found Portal built using MERN Stack (MongoDB, Express, React, Node.js).

Users can:
- Report lost items
- Report found items
- View items
- Upload images

## 🚀 Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB

## 📂 Project Structure
- client → React frontend
- server → Node.js backend

## ⚙️ How to Run

### Backend
cd server
npm install
node server.js

### Frontend
cd client
npm install
npm start
